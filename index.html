tomatotasktracker-desktop

is your personal simple task tracker with added "Pomodoro" technique

About

The manager was created to simplify the process of managing tasks. Unlike complex systems, here you will not find anything other than simply counting time.

An added bonus is a time management method "Pomodoro". The program will notify you of the expiration time periods assigned for work and leisure. By default, time to work is 25 minutes to rest is 5 minutes. These values can be changed in the program settings.

Features

  • the hierarchical structure of tasks;
  • direct and countdown timer;
  • automatic saving of the project;
  • text and audio notification about the end time periods assigned for work and rest;
  • configure time periods assigned for work and rest

    • F.A.Q.

    Q: The start button is not active. What should I do?
    A: Select the required task and click "Activate" in the "Task" menu.
